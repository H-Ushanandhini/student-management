# Student Management System

A web application built with Flask and MySQL to manage student records.

## Technologies Used
- Python
- Flask
- MySQL
- HTML (Jinja2 Templates)

## Features
- Add new student (name and email)
- View all students
- Edit student details
- Delete student record

## Project Files
- `app.py` — main Flask application
- `db.py` — database connection
- `templates/` — HTML pages

## How to Run
1. Install Flask:
pip install flask

2. Install MySQL connector:
pip install mysql-connector-python

3. Run the app:
python app.py

4. Open browser:
http://localhost:5001
